# 🐾 purr-focus

A soft, cat-themed productivity timer built for gentle focus sessions.
No harsh alarms. No pressure. Just you, your cat, and a cozy pomodoro.

## ✨ features

- 🌸 soft pomodoro timer with customizable sessions
- 😴 break screen with sleeping cat animation
- 🌧 ambient sounds — rain, café, night lo-fi
- 📝 notes & checklist page for gentle to-dos
- 🎨 pastel pink & yellow hand-drawn aesthetic

## 🛠 tech stack

- React + Vite
- Tailwind CSS v4
- Caveat & Nunito (Google Fonts)

## 🚀 getting started

git clone https://github.com/deep-shikha124/purr-focus.git
cd purr-focus
npm install
npm run dev
